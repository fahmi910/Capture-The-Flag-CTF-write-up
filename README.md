# 🧩 CTF Write-ups

This repository contains my Capture The Flag (CTF) write-ups, documenting my learning journey in cybersecurity through hands-on challenges.

I started participating in CTF challenges as part of my interest in cybersecurity, exploring domains such as web exploitation, cryptography, and forensics. This repository serves as both a personal knowledge base and a platform to document and share my approach to solving challenges.

The write-ups included here come from a combination of CTF competitions and training platforms such as picoCTF. Some solutions are completed individually, while others are collaborative efforts with my teammates. Where applicable, proper credit is given, and these write-ups reflect both personal and shared learning experiences.

---

## Purpose

- Document my learning and problem-solving process
- Improve my technical and analytical skills in cybersecurity
- Share write-ups to help others understand CTF challenges
- Build a portfolio of practical cybersecurity experience
